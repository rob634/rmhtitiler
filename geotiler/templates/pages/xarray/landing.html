Available Endpoints

Endpoint Description
GET /xarray/variables List all variables in the dataset
GET /xarray/info Get variable metadata (dims, shape, dtype)
GET /xarray/tiles/{tms}/{z}/{x}/{y} XYZ map tiles
GET /xarray/{tms}/tilejson.json TileJSON for web maps
GET /xarray/{tms}/map Interactive map viewer
GET /xarray/point/{lon},{lat} Value at point

Common Query Parameters

Parameter Description
url Zarr store or NetCDF URL (required)
variable Variable name to visualize
datetime Time index or datetime string
colormap_name Colormap (viridis, plasma, inferno, magma, etc.)
rescale Min,max value scaling (e.g., 0,100)
decode_times Set to false to disable time decoding
