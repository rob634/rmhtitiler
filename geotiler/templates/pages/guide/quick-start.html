Quick Start

Make your first API call in 5 minutes

Prerequisites

  • Python 3.8+ with requests library
  • An authentication token (see Authentication guide)

Step 1: Get COG Information

The simplest API call — get metadata about a Cloud Optimized GeoTIFF:

import requests

# Public COG - no auth needed for this example
url = "https://your-titiler-instance/cog/info"
params = {
    "url": "https://sentinel-cogs.s3.us-west-2.amazonaws.com/sentinel-s2-l2a-cogs/36/Q/WD/2020/7/S2A_36QWD_20200701_0_L2A/TCI.tif"
}

response = requests.get(url, params=params)
info = response.json()

print(f"Bounds: {info['bounds']}")
print(f"CRS: {info['crs']}")
print(f"Width x Height: {info['width']} x {info['height']}")

Step 2: Query a Point Value

Extract the pixel value at a specific coordinate:

# Query point value
url = "https://your-titiler-instance/cog/point/36.8,-1.3"
params = {
    "url": "https://sentinel-cogs.s3.us-west-2.amazonaws.com/sentinel-s2-l2a-cogs/36/Q/WD/2020/7/S2A_36QWD_20200701_0_L2A/TCI.tif"
}

response = requests.get(url, params=params)
data = response.json()

print(f"Values at point: {data['values']}")

Step 3: Get a Tile URL

Generate a TileJSON for use in web maps:

# Get TileJSON
url = "https://your-titiler-instance/cog/WebMercatorQuad/tilejson.json"
params = {
    "url": "https://sentinel-cogs.s3.us-west-2.amazonaws.com/sentinel-s2-l2a-cogs/36/Q/WD/2020/7/S2A_36QWD_20200701_0_L2A/TCI.tif"
}

response = requests.get(url, params=params)
tilejson = response.json()

print(f"Tile URL template: {tilejson['tiles'][0]}")
print(f"Bounds: {tilejson['bounds']}")
