Web Developers Guide

Build maps with MapLibre GL JS

Key Benefit MapLibre GL JS is 30kb vs 800kb for proprietary SDKs. Standards-based skills are portable.

What You Can Build

Raster Tile Maps

Display COG imagery as XYZ tiles with dynamic styling

Vector Features

Query and display PostGIS data as GeoJSON or vector tiles

Quick Example

// MapLibre GL JS - Display COG tiles
const map = new maplibregl.Map({
  container: 'map',
  style: {
    version: 8,
    sources: {
      'flood': {
        type: 'raster',
        tiles: [
          'https://titiler/cog/tiles/WebMercatorQuad/{z}/{x}/{y}?url=...'
        ],
        tileSize: 256
      }
    },
    layers: [{
      id: 'flood-layer',
      type: 'raster',
      source: 'flood'
    }]
  },
  center: [29.8, -2.0],
  zoom: 8
});

Available Tile Endpoints

Endpoint Format Use Case
/cog/tiles/{tms}/{z}/{x}/{y} PNG/WebP Raster imagery (COGs)
/xarray/tiles/{tms}/{z}/{x}/{y} PNG/WebP Zarr/NetCDF data
/searches/{id}/tiles/{tms}/{z}/{x}/{y} PNG/WebP STAC search mosaics
/vector/collections/{id}/tiles/{tms}/{z}/{x}/{y} MVT Vector tiles from PostGIS
