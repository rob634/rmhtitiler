Technology Stack

Open source libraries powering this API

This application is built on exceptional open source geospatial libraries, primarily developed by Development Seed.

Core Libraries

Library Purpose Links
TiTiler Dynamic tile server for Cloud Optimized GeoTIFFs (COGs) and other raster formats. Powers the /cog and /xarray endpoints. GitHub | Docs
TiPG OGC Features API and Vector Tiles from PostgreSQL/PostGIS. Powers the /vector endpoints. GitHub | Docs
stac-fastapi STAC API implementation for searching and browsing geospatial catalogs. Powers the /stac endpoints. GitHub | Docs
pgSTAC PostgreSQL backend for STAC catalogs. Enables dynamic mosaic tiling from STAC searches. GitHub | Docs
rio-tiler Rasterio-based library for reading and tiling geospatial raster data. GitHub | Docs

Acknowledgments

Special thanks to Development Seed for creating and maintaining these libraries that make modern geospatial APIs possible.

Development Seed

Development Seed builds open technology and data products to solve global challenges. Their geospatial tools power applications serving climate data, satellite imagery, and environmental monitoring worldwide.

developmentseed.org

Additional Dependencies

This application also relies on:

  • FastAPI - Modern Python web framework
  • Rasterio - Geospatial raster I/O
  • GDAL - Geospatial data abstraction library
  • xarray - N-dimensional labeled arrays
  • Zarr - Chunked, compressed array storage
